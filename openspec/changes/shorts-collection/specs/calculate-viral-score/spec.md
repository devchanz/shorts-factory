# Spec: calculate-viral-score

## ADDED Requirements

### Requirement: 바이럴 점수 산출
각 Shorts 영상에 대해 조회수, 좋아요, 댓글 수, 게시 후 경과 시간 등을 기반으로 바이럴 점수(0~100)를 계산해야 한다.

#### Scenario: 점수 계산 실행
- **WHEN** Shorts 데이터 수집이 완료되면
- **THEN** 수집된 각 영상에 대해 즉시 바이럴 점수를 계산한다.

#### Scenario: 가중치 적용
- **WHEN** 점수를 계산할 때
- **THEN** 좋아요와 댓글 수에 조회수보다 높은 가중치를 부여하여 단순히 조회수만 높은 영상보다 사용자 반응이 좋은 영상을 우대한다. (예: (좋아요/조회수) * a + (댓글/조회수) * b)

### Requirement: 최신성 반영
오래된 영상보다 최신 영상이 더 높은 점수를 받도록 보정해야 한다.

#### Scenario: 시간 감쇠(Time Decay) 적용
- **WHEN** 영상 게시일로부터 시간이 많이 경과할수록
- **THEN** 점수 산출 시 감쇠 계수를 곱하여 최신 트렌드를 반영한다.
