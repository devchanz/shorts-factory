# Spec: store-shorts-data

## ADDED Requirements

### Requirement: Shorts 메타데이터 저장
수집 및 분석된 Shorts 데이터를 데이터베이스에 저장해야 한다.

#### Scenario: 데이터 저장
- **WHEN** 바이럴 점수 계산이 완료되면
- **THEN** 각 영상의 메타데이터(ID, 제목, 썸네일, 채널정보 등)와 계산된 트렌드 정보를 `shorts` 및 `trends` 테이블에 저장한다.
- **AND** 이미 존재하는 영상(ID 중복)의 경우 최신 정보로 업데이트한다.

### Requirement: 데이터 보관 주기 관리
데이터베이스 용량 관리 및 트렌드 신선도 유지를 위해 7일이 지난 데이터는 삭제해야 한다.

#### Scenario: 오래된 데이터 삭제
- **WHEN** 새로운 데이터 저장이 완료된 후 (또는 별도 정리 작업 시)
- **THEN** 수집일로부터 7일 이상 경과한 데이터를 데이터베이스에서 영구 삭제한다.
